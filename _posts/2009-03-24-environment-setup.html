--- title: Environment setup date: '2009-03-24T00:27:00.001-07:00' author: bloggerming tags: - management modified_time: '2009-03-24T00:27:21.216-07:00' blogger_id: tag:blogger.com,1999:blog-3711184356900467447.post-5613295413197077043 blogger_orig_url: http://technoming.blogspot.com/2009/03/environment-setup.html --- Must have a environment setup so that the company can run right.
Must have a file server where commonly used files can be stored eg. slide templates.
This can also contain SOP (Standard Operating Procedures) documents.
Must have CMS (Content Management System) or DMS (Document Management System) such as knowledgetree.
Must have a bulletin board system where announcements can be made in order to reduce email flooding.
For software companies
A source control management system must be used in order to version control software.
Preferably one with changelists capabilities.
A bug tracking system must also be used and it must be used in conjunction with changelists so that bug numbers can be traced to changelists.
